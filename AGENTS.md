# Project Instructions for Codex

This is a static browser-only e-Duti XML Generator.

No backend.
No npm.
No framework.
No database.
No persistence.
No hosting requirement.

Primary quality areas:
1. Sanitization
2. Validation
3. Conditional mandatory fields
4. Attachment validation
5. XML generation
6. XML escaping
7. Download output

Use TEST_PLAN.md as the QA source.
Use SPECIFICATION.txt as the business-rule source.

Do not redesign the UI.
Do not add dependencies unless explicitly requested.
Prefer small targeted fixes.

Test by opening app/test.html and app/index.html directly in a browser.
